## OVERVIEW: WHAT IS LOCALE ACADEMY?

**Locale Academy** is an interactive learning platform (similar to OpenAI Academy) that teaches every user of Locale how to **get the most value** from their AI agents, AI-powered document management, and partnership opportunities.

**For Partners:** "Level up your service delivery with AI"  
**For Clients:** "Get maximum ROI from your AI workforce"  
**For Agents:** "Understand when/how to tap out and tag in"

**Core Value:** Pre-built, battle-tested prompts + workflows + success stories = faster onboarding + better results + higher retention.

## LOCALE ACADEMY: CORE VERTICALS

### **1. PARTNERS ACADEMY**
*For service providers using Locale Marketplace*

**Modules:**

#### Module 1: "AI Agent As Your Assistant"
- **Goal:** Use ACHEEVY to manage clients 10x faster
- **Prompts:**
  - "Client Onboarding Checklist" (what to ask new clients)
  - "Service Proposal Generator" (turn client brief into detailed proposal)
  - "Rate Optimization" (calculate your ideal pricing)
  - "Client Follow-up Automation" (stay in touch without effort)

**Example Prompt:**
```
**PROMPT:** "Client Onboarding Checklist"

You are my business assistant. A new client just booked me for [SERVICE TYPE].

Generate a checklist of 10 key questions I should ask them before starting, organized by:
1. Project Scope (timeline, deliverables, constraints)
2. Budget & Payment (budget range, payment terms, approval process)
3. Communication (preferred channels, decision-makers, escalation path)
4. Success Metrics (how do we define "done"?)

For each question, provide:
- The question itself
- Why it matters
- What answer flags a "red flag"

Format as a checklist I can use in my first call.
```

#### Module 2: "Tag-Team Model Rotation (The Wrestling Metaphor)"
- **Goal:** Understand when to "tap out" an agent and bring in a fresh one
- **Prompts:**
  - "Health Check: Is My Agent Tired?" (detect when to switch)
  - "Tag-In Protocol" (how to hand off to another model without losing context)
  - "Conflict Resolution" (if agents disagree on approach)

**Example Prompt:**
```
**PROMPT:** "Is My Agent Tired?"

I've been working with an AI agent for [X] hours on [PROJECT].

The agent has been [DESCRIBE BEHAVIOR]:
- Repeating the same suggestions
- Taking longer to respond
- Losing track of earlier decisions
- Generating lower-quality outputs

Is it time to "tap out"? If yes, how do I hand off without losing progress?

Provide:
1. Health score (0-100, where 100 = fresh agent)
2. Signs that suggest a tap-out is needed
3. Step-by-step handoff protocol (what context to preserve)
```

#### Module 3: "Profit Mechanics 101"
- **Goal:** Understand the refund reserve and how to not lose money
- **Prompts:**
  - "My Profit Breakdown" (see your per-task reserve in real-time)
  - "When Will Locale Refund Me?" (understand the conditions)
  - "Scale Without Risk" (how to grow without betting the company)

**Example Prompt:**
```
**PROMPT:** "My Profit Breakdown"

I completed [N] tasks this month.
Average task value: $[AMOUNT]

Show me:
1. My gross revenue this month
2. Platform fee (Platform keeps 1/3 of profit as reserve)
3. My net revenue (what I actually keep)
4. My refund reserve (what's held for hallucinations)
5. My safe-to-spend amount (gross - reserve)

Also: If one of my tasks failed (agent hallucinated), how much would Locale refund?
```

### **2. CLIENTS ACADEMY**
*For customers buying AI services via Locale*

**Modules:**

#### Module 1: "Get Maximum ROI from Your AI Agent"
- **Goal:** Write better briefs, get better results
- **Prompts:**
  - "Perfect Brief Generator" (how to write a task description)
  - "Success Metrics Template" (what "done" really means)
  - "Red Flags: When to Pause an Agent" (detect early failure)

**Example Prompt:**
```
**PROMPT:** "How to Write the Perfect Brief"

I want [GENERAL DESCRIPTION OF WHAT I NEED].

But I'm not sure how to describe it clearly to an AI agent.

Help me:
1. Break down my goal into 3 sub-goals
2. Define "success" (what would a perfect result look like?)

4. Identify 2 things that would make it worse (avoid these!)
5. Provide a template I can copy-paste to the agent

#### Module 2: "When to Pause / When to Continue"
- **Goal:** Know when an agent is heading for failure
- **Prompts:**
  - "Agent Health Check" (is this on track?)
  - "Mid-Course Correction" (pivot the agent if needed)
  - "Should I Ask for a Refund?" (understanding the refund guarantee)

**Example Prompt:**
```
**PROMPT:** "Is My Agent Heading for Failure?"

I'm [PERCENTAGE]% through a task. The agent:
- [BEHAVIOR #1]
- [BEHAVIOR #2]
- [BEHAVIOR #3]

Is this normal? Should I pause and ask for a different agent?

Provide:
1. Health score (red/yellow/green)
2. What these behaviors mean
3. When to pause (exact threshold)
4. How to pivot mid-task (is it allowed?)
5. When to ask for refund (criteria)
```

#### Module 3: "Build Long-Term Partnerships with Your AI Team"
- **Goal:** Create repeatable processes with consistent agents

```

I need help with [PROJECT TYPE].

- I prefer [DETAIL LEVEL] (very detailed vs. high-level)

- Tier 1: Fast, cheap, good enough

```

Feature: [FEATURE NAME]

- Time savings > 10 hours/week
- Accuracy required < 99.5% (or refund is acceptable)

Score our feature:
1. Time savings potential (0-10)

- **Goal:** Forecast revenue + understand margins

  - "CAC Payback Period" (when do we break even on acquisition?)
  - "Refund Impact Analysis" (do refunds hurt profitability?)

```

We have [N] active Partners/Clients

### **Frontend Component**

```typescript
// components/Academy.tsx
import { useState } from 'react';
import { Button, Card, Input } from '@/components/ui';

export function Academy() {
  const [selectedRole, setSelectedRole] = useState(null);
  const [selectedModule, setSelectedModule] = useState(null);
  const [selectedPrompt, setSelectedPrompt] = useState(null);

  return (
    <div className="academy-container">
      {/* Step 1: Role Selection */}
      {!selectedRole && <RoleSelector onSelect={setSelectedRole} />}

      {/* Step 2: Module Selection */}
      {selectedRole && !selectedModule && (
        <ModuleSelector role={selectedRole} onSelect={setSelectedModule} />
      )}

      {/* Step 3: Prompt Display */}
      {selectedPrompt && (
        <PromptCard
          prompt={selectedPrompt}
          onCopy={() => copyToClipboard(selectedPrompt.prompt_text)}
          onOpenInChatGPT={() => window.open(`https://chat.openai.com?text=${encodeURIComponent(selectedPrompt.prompt_text)}`)}
        />
      )}
    </div>
  );
}
```
