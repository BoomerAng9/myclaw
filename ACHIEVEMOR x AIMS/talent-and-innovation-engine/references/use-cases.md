# Use Cases

## 1. Original matrix recreation

Use the source initiative materials and preserve the original logic while modernizing labels and removing unnecessary country lock-in.

## 2. National initiative adaptation

Example pattern:
- identify the national initiative priorities
- identify sectors likely to create jobs
- map training topics and credentials to those sectors
- show likely role pathways and confidence-ranked fits

## 3. City or regional vision adaptation

Example pattern:
- identify target industries in the city or regional plan
- identify infrastructure, workforce, education, logistics, public service, or technology priorities
- build the matrix around target groups, pathways, soft skills, and role outcomes

## 4. Candidate or cohort matching

When a learner or candidate profile is supplied, use the same matrix logic to score how well the person or cohort aligns to targeted roles.

Separate this from the initiative pathway matrix itself, but show both when useful.
